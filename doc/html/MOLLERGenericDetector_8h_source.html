5//
6//MOLLERGenericDetector
7//
8// A generic detector class which could contain the following types
9// of data:
10// - ADC
11// - Can be single valued OR
12// - Can contain pulse information such as integral, amplitude and time
13// - Can be using Waveform data
14// - TDC
15// - Can be single valued (leading edge) OR:
16// - Can contain both leading and trailing edge and Time-Over-Threshold
17//
18// Organized into row, col, and layer structure (but makes no assumption about
19// number of rows, cols or layers are constant throughout).
20//
